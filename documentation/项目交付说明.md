# Amazon Seller Central Clone - 项目交付说明

## 📋 项目概述

### 项目名称
Amazon Seller Central Clone - 高保真度电商卖家中心克隆

### 项目描述
完整复制Amazon Seller Central的多步骤认证流程、市场切换、本地化仪表板和数据导入/导出功能的全栈应用。

### 技术栈
- **前端**：React 18 + TypeScript + Vite + Tailwind CSS
- **后端**：Node.js + Express.js + TypeScript
- **管理后台**：React 18 + TypeScript + Ant Design
- **数据存储**：JSON文件存储
- **部署**：本地部署，支持域名和localhost两种模式

## 🚀 核心功能

### 1. 多步骤认证系统
- **邮箱输入** → **密码验证** → **OTP两步验证**
- 完整的表单验证和错误处理
- 多语言支持（中英文）
- 会话持久化

### 2. 多市场支持
- **支持市场**：美国、日本、英国、德国、欧洲
- **货币切换**：USD、JPY、GBP、EUR
- **本地化数据**：不同市场的差异化数据展示
- **数据倍率**：美国1.5x、日本1.2x、英国1.0x、德国0.7x

### 3. 仪表板功能
- **欢迎横幅**：个性化欢迎信息
- **全局快照**：6列关键业务数据
- **商品表现表格**：可搜索、可排序的商品列表
- **左侧操作卡片**：快速操作入口
- **实时数据更新**：WebSocket支持

### 4. 库存管理
- **商品列表**：搜索、筛选、分页功能
- **状态管理**：库存状态实时更新
- **批量操作**：批量编辑商品信息
- **媒体预览**：商品图片预览功能

### 5. 业务报告
- **销售仪表板**：图表和数据可视化
- **销售快照**：关键指标展示
- **图表/表格切换**：多种数据展示方式
- **Recharts集成**：专业图表库

### 6. 多店铺管理
- **店铺切换**：无缝切换不同店铺
- **数据隔离**：每个店铺独立的数据
- **差异化展示**：不同店铺的个性化数据

## 🛠️ 部署方案

### 开发环境
```bash
# 安装依赖
npm install

# 启动开发环境
dev-tools/dev-start.bat

# 或分别启动各服务
npm run dev:frontend   # 前端
npm run dev:backend    # 后端  
npm run dev:admin      # 管理后台
```

### 客户部署
1. **打包**：运行 `build-tools/final-package.bat`
2. **分发**：将生成的zip包发给客户
3. **安装**：客户运行 `install.bat`
4. **启动**：客户运行 `smart-launcher.bat`

### 支持的启动模式

#### 本地模式 (推荐)
- **适用**：大多数用户
- **特点**：启动快速，运行稳定
- **访问**：http://localhost:3000

#### 域名模式 (高级)
- **适用**：需要自定义域名的用户
- **特点**：支持自定义域名，集成代理服务器
- **配置**：使用域名配置工具
